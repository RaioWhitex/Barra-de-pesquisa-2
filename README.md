# 🔍 Search Bar UI

Uma interface moderna de barra de pesquisa desenvolvida com foco em simplicidade, usabilidade e design minimalista. O projeto apresenta um campo de busca elegante acompanhado de um botão de ação destacado, proporcionando uma experiência intuitiva para pesquisas em aplicações web.

---

## 📸 Preview do Projeto

Interface contendo:

- Campo de pesquisa estilizado
- Ícone de busca integrado
- Botão de pesquisa destacado
- Layout moderno e centralizado
- Estrutura limpa e organizada
- Design responsivo

---

## ✨ Tecnologias Utilizadas

- **HTML5**
- **CSS3**

---

## 🎨 Características do Design

- Interface moderna e minimalista
- Campo de pesquisa amplo
- Ícone visual intuitivo
- Botão Call-To-Action (CTA)
- Bordas arredondadas
- Paleta de cores elegante
- Efeitos hover suaves
- Layout responsivo
- Excelente experiência do usuário

---

## 🚀 Objetivo do Projeto

Praticar habilidades de desenvolvimento front-end através da criação de uma barra de pesquisa moderna, explorando conceitos de formulários, alinhamento de elementos, responsividade e estilização avançada utilizando HTML e CSS.

---

## 📚 Funcionalidades

- Campo para inserção de texto
- Botão de pesquisa personalizado
- Ícone de busca integrado
- Estrutura reutilizável para diferentes projetos
- Fácil personalização de cores e dimensões
- Compatibilidade com diversos dispositivos

---

## 📱 Responsividade

O projeto foi desenvolvido para proporcionar uma experiência consistente em diferentes tamanhos de tela:

- Desktop
- Tablet
- Smartphone

## 💡 Aprendizados

Durante o desenvolvimento deste projeto foram praticados conceitos como:

- Estruturação de formulários
- Flexbox
- Responsividade
- Inputs personalizados
- Botões interativos
- UI/UX Design
- Estilização avançada com CSS3
